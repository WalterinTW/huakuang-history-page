# AGENTS.md

## Project goal
Build a single-page static website for Huaguang Temple history section in Traditional Chinese.

## Design reference
Use the following site as visual inspiration only:
https://fagushan.ddm.org.tw/

Do not copy the source code or duplicate the site layout exactly.
Instead, capture these qualities:

- calm, dignified, Buddhist cultural atmosphere
- clean whitespace and readable typography
- elegant and restrained color palette
- soft earth / beige / stone / wood-like tones
- large banner image with title overlay
- content sections with clear hierarchy
- image-forward but text remains highly readable
- desktop and mobile responsive
- avoid flashy animations and overly commercial styling

## Visual direction
- tone: serene, respectful, historical
- typography: readable Traditional Chinese, not decorative
- layout: modern but restrained
- cards and timeline elements are allowed
- use subtle dividers, soft background sections, gentle shadows only if needed

## Page scope
For now, only build one page:
- 歷史沿革
